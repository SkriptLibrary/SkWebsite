# Modules

You can **install** modules with <br> ```/sklibrary modules install (moduleName - you see a list of all modules)``` <br> or **remove** modules with <br> ```/sklibrary modules remove (moduleName - you see a list of all your installed modules)```<br>

---

# Addons
Addons will be **automaticly installed** with the module.

If the module need a _Skript Addon_, there will be a pop-up to **install it automaticly** & **restart** the server.
